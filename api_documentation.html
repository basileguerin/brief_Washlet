Chiottes API Documentation

Endpoint: /chiottes/add

Method: POST

Description: Ajoute un nouveau produit à la base de données.

Request body:

{
  "titre": "string",
  "ref": "string",
  "dimension": "string",
  "couleur": "string"
}

Endpoint: /chiottes/delete_all

Method: DELETE

Description: Supprime tous les produits de la base de données.

Endpoint: /chiottes/delete/{chiotte_id}

Method: DELETE

Description: Supprime un produit de la base de données en fonction de son identifiant.

Endpoint: /chiottes/update/{chiotte_id}

Method: PUT

Description: Met à jour les informations d'un produit en fonction de son identifiant.

Request body:

{
  "titre": "string",
  "ref" : "string",
  "dimension": "string",
  "couleur": "string"
}

Endpoint: /chiottes

Method: GET

Description: Renvoie la liste de tous les produits.

Endpoint: /chiottes/add_all

Method: POST

Description: Ajoute tous les produits à la base de données.
